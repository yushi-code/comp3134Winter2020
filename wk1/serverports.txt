1) What is the definition of server port?

The answer is: A port number is a way to identify a specific process to which an Internet or other network message is to be forwarded when it arrives at a server.

2) What is server port range?

The answer is 0 to 65535

3) How many bits does number above represent?
65536 is 2^16, so it's 16 bits
